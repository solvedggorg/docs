**zig-libsql** is a Zig adapter for **libSQL** / SQLite-compatible databases, built for solved.gg toolchains written in pure(as) Zig ([rusty](/rusty), [scripty](/scripty), [hasky](/hasky), [clerk-zig](/clerk-zig), …).

- **Local:** vendored SQLite amalgamation compiled by Zig (not system `libsqlite3`)
- **Remote:** pure Zig **Hrana over HTTP** (JSON `v3/pipeline`)
- **No cargo** on the default path. If a Rust bridge is ever required, build it with [rusty](/rusty).

## Requirements

- Zig **0.16.x**
- libc (linked for the local engine)

## Status matrix

| Capability | Status | Notes |
| --- | --- | --- |
| Local file DB | **MVP** | Vendored SQLite amalgamation |
| In-memory DB | **MVP** | `:memory:` |
| Prepare / bind / step | **MVP** | Positional + named binds |
| Transactions | **MVP** | begin / commit / rollback |
| Batch | **MVP** | Local txn + remote Hrana batch |
| Remote Hrana HTTP | **MVP** | JSON `v3/pipeline` |
| Hrana WebSocket | later | |
| Embedded replicas | In progress | Wire + pull landed; inject next |
| libSQL SQL extensions | deferred | Default stock SQLite; opt-in `-Dengine=libsql` |
| System libsqlite3 backend | non-goal | |
| Rust C FFI default | non-goal | Optional bridge only |

## Library usage

```zig
const libsql = @import("zig_libsql");

var db = try libsql.Database.open(allocator, .{ .path = ":memory:" });
defer db.deinit();

var conn = db.connect();
try conn.exec("create table t(id integer primary key, name text);", .{});
try conn.exec("insert into t(name) values ('alice');", .{});

var stmt = try conn.prepare("select id, name from t where id = ?1;");
defer stmt.deinit();
try stmt.bindInt(1, 1);

while (try stmt.step()) |row| {
    const id = try row.int(0);
    const name = try row.text(1);
    _ = .{ id, name };
}
```

### Remote (Hrana HTTP)

```zig
var db = try libsql.Database.open(allocator, .{
    .path = "libsql://your-db.turso.io",
    .auth_token = token, // never log
    .io = io,            // required for remote
});
defer db.deinit();
var conn = db.connect();
try conn.exec("create table if not exists t(x int);", .{});
```

`libsql://` and `wss://` map to `https://` for the HTTP pipeline.

## Design layers

| Layer | Implementation |
| --- | --- |
| Public API | Idiomatic Zig (`Database`, `Connection`, `Statement`, `batch`) |
| Local engine | `vendor/sqlite3.c` (default) or libSQL amalgamation via `-Dengine=libsql` |
| Remote | Hrana over HTTP JSON |
| Replicas | Wire + pull landed; inject in progress |
| Rust | Optional rusty cdylib for classic replica sync (`-Denable-rust-bridge`) |
