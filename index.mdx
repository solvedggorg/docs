Tooling for modern language ecosystems — Rust, JavaScript, and Go — built in Zig and shipped by [iResolved, LLC](https://solved.gg).

Use the **product switcher** in the navbar to jump between docs for each tool.

<CardGroup cols={3}>
  <Card title="rusty" icon="gears" href="/rusty">
    Cargo + rustup, rebuilt in Zig for Linux monorepos. Content-addressable stores, native resolve/build/test, up to ~400× faster than cargo on large workspaces.
  </Card>
  <Card title="scripty" icon="code" href="/scripty">
    A JS runtime and project manager with Node.js, Deno, and Bun APIs — plus formatting and linting — written in Zig.
  </Card>
  <Card title="wargo" icon="cube" href="/wargo">
    A next-gen Go workspace project manager, written in Zig.
  </Card>
