**depotsdk-go** es un cliente Go para la API [Depot](https://depot.dev). Envuelve stubs de servicio compatibles con Connect/gRPC para que puedas ver una lista de proyectos, gestionar compilaciones y llamar a otras API de la organización Depot desde Go.

## Para qué sirve depotsdk-go

- **Acceso tipado a Depot y API** desde los servicios y las CLI de Go
- **Autenticación con token de portador** (tokens a nivel de organización o de compilación)
- **Solicitudes al estilo Connect** a través del Depot HTTP API

La documentación canónica del módulo y del paquete evoluciona con el repositorio; cuando los canales de instalación difieran del empaquetado Depot del upstream, esta página indicará explícitamente la ruta del módulo **solved.gg**.

## Instalación (cuando se publique)

Apunta `go get` a la ruta del módulo que ofrece el repositorio **solvedggorg/depotsdk-go** una vez que lo utilices como dependencia. Hasta que las notas de publicación aparezcan aquí, clona o sustituye desde el remote de la organización y sigue las instrucciones del repositorio `README`.

```bash
# Example shape — confirm module path in go.mod before relying on it in CI
go get github.com/solvedggorg/depotsdk-go@latest
```

Si tu copia del código sigue usando una ruta de módulo diferente, usa esa ruta hasta que se cambie el nombre.

## Autenticación

| Variable | Descripción |
| --- | --- |
| `DEPOT_TOKEN` o `DEPOT_API_TOKEN` | Token de organización o de compilación |
| `DEPOT_API_URL` | URL base de API (por defecto Depot si no está configurada) |

Usa `Authorization: Bearer` con un token de organización (o un token de compilación para llamadas dentro del ámbito de la compilación).
