## Production hosts

| --- | --- |

## Local development hosts

Typical local defaults when running the monorepo:

| Surface | URL |
| --- | --- |
| Web | http://localhost:5173 |
| API | http://127.0.0.1:8787 |

Local callbacks use the **same path segments** as production with localhost hosts, for example:

- `http://localhost:5173/linear/auth/callback`
- `http://localhost:5173/github/auth/callback`
- `http://localhost:5173/github/setup`

OAuth apps must list those callback URLs if you test OAuth locally.

## Clerk

Hosted auth uses the solved.gg Clerk instance (custom domain such as `clerk.solved.gg` in production). Organizations must be enabled.
